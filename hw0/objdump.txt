The file format for the exectuable is elf64-x86-64.

It is compiled for the architecture: i386:x86-64.

Some segment/sections are .interp, .init, .plt, .text, .finim, .dynamic, and .data

The .text segment contains main.  The address is 0x40067d.

There is no stack or heap segment.  These segments aren't created until the executable is run.
